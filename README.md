Week 8 (Activity 1): Proximity & Distance Analytics

This project analyzes delivery/pickup point distributions using unsupervised machine learning to improve logistics efficiency.

Key Deliverables
- Clustering Analysis: Implementation of K-Means (partitioning) and DBSCAN (density-based) clustering.
- Visualizations: Interactive map identifying cluster centroids and delivery zones.
- Operational Summary: CSV summaries containing cluster stats, including center locations and service radii.

How to Run
1. Ensure Python 3.x is installed with the required libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`, `folium`.
2. Open `Proximity Distance Analytics.ipynb` in Jupyter Notebook or Google Colab.
3. Run the cells sequentially to reproduce the clustering results and map generation.
